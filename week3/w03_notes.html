Chapter 6 The Document Object Model (DOM)

Document Object Model - represents everything on an HTML page as
nodes. From tags to attributes.

The DOM treats everything on a web page as a node. HTML tags, the
text inside these tags, even the attributes of a tag are all
nodes. The HTML tag is the root node, and every other part of the
document is a child node of this.

const body = document.body;

body.nodeType;
<< 1

Codes and their type
1 : element, 2 : attribute, 3 : text, 8 : comment, 9 : body

Setting an Element's Attribute
wonderWoman.setAttribute('class', 'hero');

Instead of having to always monitor when an event occurs, event
listeners send notification alerts to the program. Some common
listeners are mouse listeners or keyboard listeners.

Handling Events

<!doctype html>

<p> Drag the bar to change its width:</p>

</div>
<script>
let lastX; // Tracks the last observed mouse X position
let bar = document.querySelector("div");
bar.addEventListener("mousedown", event => {
if (event.button == 0) {
lastX = event.clientX;
window.addEventListener("mousemove", moved);
event.preventDefault(); // Prevent selection
}
});

function moved(event) {
if (event.buttons == 0) {
window.removeEventListener("mousemove", moved);
} else {
let dist = event.clientX - lastX;
let newWidth = Math.max(10, bar.offsetWidth + dist);
bar.style.width = newWidth + "px";
lastX = event.clientX;
}
}
</script>
